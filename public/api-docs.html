üîë Authentication

API Key Authentication: All API requests require an API key in the request headers.

To authenticate your requests:

  1. Go to API Keys page in your AvatarLab dashboard
  2. Click Create API Key
  3. Configure permissions and scope (specific avatar or all avatars)
  4. Copy your API key (you'll only see it once!)

Include the API key in all requests:

x-api-key: pk_live_your_api_key_here
‚ö†Ô∏è Security Warning: Keep your API keys secret! Never commit them to version control or share them publicly.

üåê API Endpoints

Base URL

https://your-project-id.supabase.co/functions/v1

POST /avatar-chat

Send a message to an avatar and get a response. Includes RAG knowledge base, memories, and trained prompt versions.

Request Headers

Header Value Required
x-api-key Your platform API key Required
Content-Type application/json Required

Request Body

Parameter Type Description Required
avatar_id string (UUID) The ID of the avatar to chat with Required
message string The user's message Required
conversation_history array Previous messages for context (max 30) Optional
model string OpenAI model (default: gpt-3.5-turbo) Optional

Example Request

{
  "avatar_id": "123e4567-e89b-12d3-a456-426614174000",
  "message": "Tell me about your background",
  "conversation_history": [
    {
      "role": "user",
      "content": "Hello!"
    },
    {
      "role": "assistant",
      "content": "Hi! How can I help you today?"
    }
  ],
  "model": "gpt-4o-mini"
}

Example Response

{
  "success": true,
  "avatar_id": "123e4567-e89b-12d3-a456-426614174000",
  "message": "I grew up in Malaysia and developed a passion for technology...",
  "metadata": {
    "model": "gpt-4o-mini",
    "knowledge_chunks_used": 3,
    "memories_accessed": 5
  }
}

üîó n8n Integration Guide

Quick Setup: Follow these steps to connect your avatar with n8n and WhatsApp

Step 1: Get Your API Key

  1. Login to AvatarLab
  2. Navigate to API Keys page
  3. Click Create API Key
  4. Name it (e.g., "n8n WhatsApp Bot")
  5. Select the avatar you want to connect
  6. Choose permissions: chat, config
  7. Copy your API key

Step 2: Create n8n Workflow

  1. Trigger Node: WhatsApp Trigger (or any trigger you prefer)
  2. HTTP Request Node: Configure as follows:
    • Method: POST
    • URL: https://your-project.supabase.co/functions/v1/avatar-chat
    • Authentication: None (use header instead)
    • Headers:
      • x-api-key: Your API key
      • Content-Type: application/json
    • Body: 
      {
  "avatar_id": "your-avatar-id",
  "message": "{{ $json.message }}",
  "conversation_history": []
}
  3. WhatsApp Response Node: Send the response back:
    • Message: {{ $json.message }}

Step 3: Test Your Workflow

  1. Activate your n8n workflow
  2. Send a WhatsApp message to your trigger number
  3. Your avatar will respond using its personality, knowledge base, and memories!
üí° Pro Tip: Store conversation history in a database to maintain context across multiple messages. Use n8n's built-in database nodes or external services like Redis.

‚ö†Ô∏è Error Handling

HTTP Status Codes

Code Description
200 Success
400 Bad Request - Missing required parameters
401 Unauthorized - Invalid or missing API key
403 Forbidden - API key lacks permission
404 Not Found - Avatar doesn't exist
500 Internal Server Error

Error Response Format

{
  "error": "Detailed error message here"
}

Common Errors

Missing API Key

{
  "error": "Missing API key. Include x-api-key header."
}

Invalid API Key

{
  "error": "Invalid or inactive API key"
}

Permission Denied

{
  "error": "API key does not have chat permission"
}

Avatar Not Found

{
  "error": "Avatar not found"
}
